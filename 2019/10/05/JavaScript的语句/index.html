  1. switch…case 分支语句中，switch 表达式的值是要恒等于（===） case 的值才能匹配，并不能自动作类型转换。

console.time(String str); //创建一个计时器，str 为计时器的名称 。放在需要检测的语句块的开头。
console.timeEnd(String str); //结束计时器，str 为计时器的名称。放在需要检测的语句块的结尾。
