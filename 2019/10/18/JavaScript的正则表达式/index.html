JavaScript 的正则表达式

定义

用于定义一些字符串的规则。
计算机可以根据正则表达式来检查一个字符串是否符合该规则，或将字符串中符合规则的部分字符串提取出来。

创建正则表达式对象

使用构造函数创建正则表达式

语法：var 变量 = new RegExp("正则表达式","匹配模式");
匹配模式（有些书也叫修饰词）有三个值：

  1. i：忽略字母大小写；
  2. g：全局匹配模式；
  3. m：多行匹配；

匹配模式可以是单独的 i、g、m，也可以是它们中的任意组合。

注意：两个参数都是字符串类型。当字符串中含有特殊字符时，要使用转义字符\进行转义。

使用直接量创建正则表达式

语法：var 变量 = / 正则表达式 / 匹配模式

注意：构造函数 与 直接量 的区别

在构造函数中，是以字符串作为参数的，.在字符串中是特殊字符，需要转义。所以在正则表达式对象是.l。而直接量中，是直接创建正则表达式对象，没有经过转义，所以在正则表达式对象是\.l。
如果正则表达式也要进行转义时，使用构造函数创建对象时，字符串中需要进行两层转义。

正则表示的使用规则

常用字符

|表示或者，两者满足一个即可。
[]里的字符也表示或者，满足任意一个字符即可。
-出现在[]中，表示从一个字符到另一个字符之间（包括两端的字符）中的任意一个字符。
常用规则：

  • [0-9]：表示0到9之间的任意一个字符。
  • [a-z]：表示任意小写字母。
  • [A-Z]：表示任意大写字母。
  • [A-z]或[A-Za-z]：表示任意字母。

^出现在[]中，表示除了，只要不满足后面的规则即可。如：非数字可以表示为[^0-9]

量词

{n}：表示前一项正好出现 n 次。
{n,m}：表示前一项正好出现 n~m次。
{,m}：表示前一项至多出现 m 次。
{n,}：表示前一项至少出现 n 次。
+：表示至少出现1次，相当于{1，}。
*：表示至少出现0次，相当于{0，}。
?：表示出现0次或1次，相当于{0，1}。

锚字符

^：以后一项开头。

$：以前一项结尾。

转义

在正则表达式中，转义字符是\。
\w：任意一个字母、数字、或下划线 _ ，相当于[A-Za-z0-9_]
\W：除了字母、数字、，相当于`[^A-Za-z0-9]\d：任意一个数字，相当于[0-9]\D：除了数字，相当于[^0-9]\s：空格\S：除了空格\b：单词边界\B`：除了单词边界

正则表达式对象的方法

test()方法

功能：检查一个字符串中，是否符合正则表达式的规则。符合返回 true ，不符合返回 false。
参数：一个字符串，表示该字符串是否符合规则。

String对象的方法（4个）与正则表达式结合使用

String 对象的方法

  1. split()功能：将字符串按照某个字符拆分成数组并返回。参数可以是一个字符，也可以是一个正则表达式对象，表示以此为分隔符。

    例题：将字符串 “1a2b3c4d5e6f7g8” 根据任意字母拆分成数组。

    根据前面学习的知识，我们知道split()方法可以拆分字符串为数组，但只能通过同一个字符拆分，显然完成不了本题的要求。其实，可以将一个正则表示式对象作为参数，进行拆分。本题是以任意字母为分隔符拆分字符串，正则表达式为/[A-z]/，如：
  1. search()在字符串中搜索是否含有指定内容。如果没有，则返回-1；如果有则返回第一次出现该内容的第一个字符的索引值。参数：可以是指定的字符、字符串、正则表达式。
注意：search()方法的匹配模式不可以是全局g。就算是全局模式，也只会返回第一次出现的索引，所以全局模式在此方法中没有用处。

  1. match()可以根据正则表达式，从一个字符串中将符合条件的内容提取出来，并将内容存储在一个数组中，返回。
    默认情况下，match() 找到第一个符合要求的内容，就会停止检索了，返回数组。可以使用全局g模式，提取出所有符合条件的内容。

  2. replace()功能：返回将字符串中的指定内容替换为新的内容。
    参数：
    第一个参数：表示指定内容。可以是一个字符串，也可以是正则表达式对象。
    第二个参数：表示新的内容。如果新内容为空字符串，表示删除指定内容。

示例：去除输入的字符串前后两端的空格。
