远程仓库GitHub项目上传或下载

前提：

  1. 有一个GitHub的账号，没有可以先注册一个账号。
    相关学习：
    注册GitHub账号和创建仓库
  2. 集成git。
    相关学习：集成git

IDEA将项目上传到远程仓库GitHub

  1. 登录账号后，在GitHub上创建一个仓库。

    仓库创建成功后，是一个空仓库，如图：
  2. 在IDEA上连接仓库。
    push:推，指上传代码；pull:拉，下载代码。
  3. 指定仓库。

  1. 验证信息。
    有可能是需要登录GitHub，也有可能是输入ssh的密钥。下面是输入ssh密钥：

    直接推送。
  2. 推送成功。

除了在推送时，验证GitHub账号，还可以提前将GitHub引入IDEA中

在settings界面中，引入GitHub账号，如图：

引入账号成功，邮箱会收到邮件提醒，显示如下：

更新代码

  1. 更改项目结构或代码。
    比如：去掉web目录，在addUser中添加红框中的代码。
  2. 同步到GitHub上。

    在这个过程中，可以书写些说明性的描述，push
  3. 代码更新同步后。
    在GitHub仓库中查看，少了web目录，且添加了红框中的代码。

IDEA将项目从远程仓库GitHub下载

  1. 在GitHub仓库中，选定一个要下载的项目，把仓库地址复制下来。
  2. 集成git和将github账号进入IDEA后，从版本控制系统中引入项目。
  3. 填写刚才复制的仓库地址，项目存放的根目录，clone。
