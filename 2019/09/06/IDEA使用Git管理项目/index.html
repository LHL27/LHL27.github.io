IDEA使用Git管理项目

当和团队一起开发项目或完成了一个项目后，我们可能要使用一些工具对项目进行管理，有利于代码共享。

使用Git管理项目

Git管理项目是在本地管理的，即在开发项目本机进行管理。

  1. 在本计算机上安装Git软件。
    直接百度找到Git官网，下载相应的版本进行解压即可。解压过程中，记住文件路径。
  2. 在IDEA中引入Git。
    在settings界面中，设置Git的路径，OK.
  3. 在IDEA中创建仓库。

    为仓库建立路径
  4. 仓库建立成功。
    在仓库目录中，如果多了.git文件，说明已建立连接，仓库建立成功。


    成功建立仓库后，可以在IDEA上看到项目的文件变红了。
  5. 将IDEA上的项目分享到仓库
    将代码添加到了Git的缓存区中，可以发现，文件变成绿色了。


    将文件上传。

    书写一些描述上传信息，可以省略。


    这是可以看到一些文件上传成功的提示，文件也由绿色变成黑色。
